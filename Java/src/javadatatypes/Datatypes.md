# Java Datatype

## Datatype Declaration Syntax

```java
datatype variableName;
```

## Datatype Initialization Syntax

```java
datatype variableName = value;
```

# Java Datatypes

# Primitive Datatypes 

## byte

```java
byte variableName = value;
```

## short

```java
short variableName = value;
```

## int

```java
int variableName = value;
```

## long

```java
long variableName = valueL;
```

## float

```java
float variableName = valueF;
```

## double

```java
double variableName = value;
```

## char

```java
char variableName = 'value';
```

## boolean

```java
boolean variableName = value;
```

# Reference Datatypes

## String 

```java
String stringVariable = new String("StringValue");
```

## Arrays

```java
int arrayName[] = { value 1, value 2, value 3 };
```
