# Java PrintStream Class

# PrintStream Class Methods

## println()

### System.out.println()

- Prints the given text.
- Then moves the cursor to a new line (adds a line break).
- So the next output will start on the following line.

## print()

### System.out.print()

- Prints the given text.
- Does not move the cursor to a new line.
- So the next output will continue on the same line.
